# IDEA离线安装插件

> 2024年6月19日11:22:32

## 一、访问官网下载插件

* 官网地址：[JetBrains Marketplace](https://plugins.jetbrains.com/)
* 搜索具体插件，打开插件的详情页面。
* 点开`Versions`，下滑页面，选择自己的版本(第二列是IDEA版本)，然后点击旁边的下载
* 下载文件一般是zip格式，使用U盘装好，转到没有网络的电脑上。

## 二、从本地安装插件

* 在IDEA打开以下页面

**File--> Settings--> Plugins-->(点击小齿轮) install plugin from disk**

* 选择刚转过来的插件即可
