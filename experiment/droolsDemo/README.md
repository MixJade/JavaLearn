## 一、概述

本项目是为了测试Drools和Swagger(或者叫Knife4j)的使用

* 如果IDEA不识别`drl`文件，可以去下载插件`Drools`，这是`JBoss`开发的插件.
* 这里的drl使用log来打印执行了那些规则(尽量不用System.out.println)

- 添加测试：Valid注解的使用
