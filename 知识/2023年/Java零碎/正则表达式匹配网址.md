我想要提取里面的网站名称、网址、简介，并将之输出为html的标签。

可以用Java这样写：

```java
public class 正则匹配网址 {
    public static void main(String[] args) {
        File file = new File("100值得拥有(1).txt");
        try (BufferedReader br = new BufferedReader(new FileReader(file))) {
            String line;
            StringBuilder sb = new StringBuilder();
            while ((line = br.readLine()) != null) {
                sb.append(line).append("\n");
            }
            // System.out.println(sb);
            // 开始正则，匹配网站名称
            List<String> matches = new ArrayList<>();
            String regex = "、(.*?) 网址：";
            Pattern p = Pattern.compile(regex);
            Matcher m = p.matcher(sb);
            while (m.find()) {
                matches.add(m.group(1));
            }
            // 再来正则，匹配所有的http或者https链接
            List<String> urls = new ArrayList<>();
            Pattern p2 = Pattern.compile("(https?://[^\\s]+)");
            Matcher m2 = p2.matcher(sb);
            while (m2.find()) {
                urls.add(m2.group(1));
            }
            // 又是正则，匹配所有的简介
            List<String> info = new ArrayList<>();
            Pattern p3 = Pattern.compile("http[s]?://[^\\s]*\\s*\\n([^\\n]*)");
            Matcher m3 = p3.matcher(sb);
            while (m3.find()) {
                info.add(m3.group(1));
            }
            if (matches.size() == urls.size() && urls.size() == info.size()) {
                StringBuilder ab = new StringBuilder();
                for (int i = 0; i < matches.size(); i++) {
                    String format = String.format("<li>%s、<a href=\"%s\">【%s】</a>%s</li>", i + 1, urls.get(i), matches.get(i), info.get(i));
                    ab.append(format).append("\n");
                }
                System.out.println(ab);
            } else {
                System.out.println("他们长度不同");
                System.out.println("网站名称列表长度：" + matches.size());
                System.out.println("网站链接列表长度：" + urls.size());
                System.out.println("网站简介列表长度：" + info.size());
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```
